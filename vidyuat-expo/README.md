# ⚡ Vidyut Expo - React Website

Central India's Biggest Electrical Expo 2026 - Official Website

## Features

- **Homepage** with countdown timer, hero banner, and registration
- **About Page** with expo mission, vision, and organizer details
- **Exhibitors Page** with categories and stall booking form
- **Gallery Page** for photos and videos
- **Contact Page** with venue details and contact information
- **Visitor Registration Form** with WhatsApp integration
- **Stall Booking Form** for exhibitors
- **Responsive Design** for all devices
- **React Router** for navigation

## Installation & Setup

1. Navigate to the project directory:
   ```bash
   cd vidyuat-expo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
